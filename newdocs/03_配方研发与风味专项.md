# 配方研发与风味专项

本章节专注于将原料科学转化为具体的配方，重点攻克抹茶与巧克力两大核心口味的研发难点，并提供故障排查指南。

---

## 第一部分：配方设计逻辑 (Recipe Design Logic)

### 1. 逆向设计法 (Reverse Engineering)
不要先想配方，要先想“结果”。按照以下五步逻辑进行设计：

1.  **设定感官目标**：
    *   **硬度**：挺拔（Llaollao 风格） -> 锁定 **固形物 > 33%**。
    *   **温度感**：浓郁温暖 -> 锁定 **高 Overrun + PAC 240-250**。
    *   **风味释放**：停留回甘 -> 锁定 **脂肪 6%-8%**。
2.  **搭建不可变基石**：
    *   **Base 50**：固定 50g。
    *   **脱脂奶粉 (SMP)**：抹茶 40g / 巧克力 30g（留空间给粉体）。
    *   **液体基底**：先假定全是牛奶。
3.  **计算风味核心**：
    *   **抹茶**：20g 抹茶粉。
    *   **巧克力**：40g 可可粉 + 80g 奶油（补脂肪）。
4.  **填补平衡空缺**（最关键）：
    *   计算 PAC 缺口（目标 250）。
    *   **蔗糖**：填补主甜度（如 130g）。
    *   **Dextrosa**：填补剩余 PAC 缺口（如 30g），调节软硬度不增过甜。
    *   **转化糖浆**：巧克力专用，增加“糯感”。
5.  **微调质地修正**：
    *   总固形物不足？加 **麦芽糊精** 填充。

### 2. 甜度曲线设计
*   **前调 (Ataque)**：蔗糖，直接爆发。
*   **中调 (Cuerpo)**：Dextrosa，清凉感中和腻。
*   **后调 (Final)**：乳糖/转化糖，悠长回甘。
*   **技巧**：
    *   **抹茶**：需转化糖/蜂蜜包裹尾段苦味。
    *   **巧克力**：加少量 **海盐**，放大甜味立体感。

## 第二部分：抹茶风味专项 (Matcha Formulation)

抹茶软冰的难点在于：**涩味抑制**、**色泽保持**、**颗粒感消除**。

### 1. 核心难点与对策
*   **苦涩控制**：
    *   **错误**：拼命加蔗糖压苦 -> 导致过甜且太软。
    *   **正确**：利用 **脂肪包埋**（7-8% 乳脂）+ **转化糖/蜂蜜** 托底。
*   **氧化变色**：
    *   **物理遮挡**：给料缸加不透明盖子。
    *   **快速周转**：少量多次投料，2-3小时内卖完。
    *   **避酸**：严禁加入酸性辅料（如柠檬汁），会导致颜色发黄。
*   **颗粒感/沉淀**：
    *   **预分散工艺**：必须做“抹茶浆”。

### 2. 抹茶配方模型 (甜口抹茶)
*(总量：1020g)*

| 原料 | 重量 | 作用 |
| :--- | :--- | :--- |
| **Leche Entera** (全脂奶) | 645 g | 溶剂 |
| **Nata 35% MG** (奶油) | 100 g | 润滑、抑制苦涩（较高比例） |
| **Leche en Polvo** (SMP) | 50 g | 骨架 |
| **Base 50 Soft** | 50 g | 稳定、乳化 |
| **Sacarosa** (蔗糖) | 120 g | 主甜度 |
| **Dextrosa** (葡萄糖粉) | 20 g | 软化质地 |
| **Maltodextrina** (麦芽糊精) | 15 g | 增加挺立度 |
| **Té Matcha** (抹茶粉) | 20 g | 风味核心 (约 2%) |

### 3. 关键工艺细节
*   **干粉预拌**：所有粉类先干混，防结块。
*   **抹茶浆制作 (关键)**：
    *   取 20g 抹茶粉 + 60g 温牛奶（从总量扣除）。
    *   用茶筅或均质机打成浓稠酱。
    *   最后混入大缸。
*   **高剪切均质**：必须使用手持均质机 (Túrmix) 高速剪切。

## 第三部分：巧克力风味专项 (Chocolate Formulation)

巧克力是软冰机的“杀手”。难点在于：**堵机风险**、**甜腻闷香**、**油水分离**。

### 1. 原料选择策略
*   **主力军**：**22/24 碱化可可粉 (Dutch Processed)**。
    *   含脂量高（22-24%），风味醇厚，颜色深红棕，不像天然粉那么酸涩。
    *   **必须碱化**：酸度低，适合甜口与奶基底。
*   **辅助军**：**Nata 35%**（补脂肪，但要克制）。
*   **调节剂**：**转化糖浆**（增亮、增糯）。
*   **禁忌**：**纯巧克力块/考维曲 (Cobertura)**。
    *   含大量硬质可可脂，在软冰机低温下易结晶卡死搅拌轴。新手慎用。

### 2. 平衡调整 (与抹茶的区别)
*   **减法**：减少 **Nata** (80g vs 100g) 和 **SMP** (30g vs 50g)。因为可可粉自带脂肪和高固形物。
*   **加法**：大幅提高 **POD** (21-23%) 和 **Dextrosa** (30g)。因为可可脂硬，需要更多 Dextrosa 软化质地。

### 3. 巧克力配方模型 (浓郁黑巧)
*(总量：1000g)*

| 原料 | 重量 | 变化分析 |
| :--- | :--- | :--- |
| **Leche Entera** | 640 g | 溶剂 |
| **Nata 35% MG** | 80 g | 🔽 减少！给可可脂腾位置 |
| **Leche en Polvo** | 30 g | 🔽 大幅减少！防太干 |
| **Base 50 Soft** | 50 g | 稳定 |
| **Sacarosa** | 130 g | 🔼 增加！对抗苦味 |
| **Dextrosa** | 30 g | 🔼 增加！软化硬质可可脂 |
| **Cacao en Polvo** | 40 g | 22/24 碱化粉 |
| **转化糖浆 (可选)** | +20g | 增加布朗尼口感 (替代部分蔗糖) |

### 4. 关键工艺细节
*   **热混合 (Mezcla en Caliente)**：
    *   强烈推荐将部分牛奶加热到 **65°C**，先溶解可可粉。
    *   热量能让可可粉充分吸水，让可可脂融化并被乳化，成品更黑亮。
*   **均质死角**：确保液面上浮粉全部卷入。

## 第四部分：故障排查指南 (Troubleshooting)

### A. 口感类

**1. 冰渣感 / 粗糙 (Icy / Coarse)**
*   **原因**：水太多（固形物<30%）、糖不够（PAC太低，冻结太快）。
*   **解法**：增加 SMP/麦芽糊精（补固形物）；增加 Dextrosa（提PAC，抑冰晶）。

**2. 颗粒感 / 粉感 (Powdery)**
*   **原因**：粉体分散失败。
*   **解法**：过筛 -> 做浆/糊 -> 高剪切 -> 老化。

**3. 砂感 (Sandy)**
*   **原因**：乳糖结晶（SMP过高 >10%）。
*   **解法**：限制 SMP 用量，改用麦芽糊精。

### B. 结构与融化类

**1. 出水 / 融化快 (Melting Fast / Wet)**
*   **原因**：PAC 太高（糖太多/果糖）、乳化不良。
*   **解法**：减少蔗糖/果糖；检查 Base 用量；增强剪切。

**2. 塌陷 / 站不住 (Collapsing)**
*   **原因**：气泡膜破裂（脂肪破乳/SMP不足/Overrun过高）。
*   **解法**：增加 Nata（脂肪网）；增加 SMP/Base；降低 Overrun。

### C. 机器与工艺类

**1. 冻缸 / 堵机 (Freeze-up)**
*   **原因**：PAC 太低（太水/糖少）、脂肪破乳（黄油化）。巧克力最常见。
*   **解法**：增加 Dextrosa/蔗糖（提PAC变软）；减少 Nata/可可脂；关机解冻。

**2. 油水分离 / 油感 (Greasy)**
*   **原因**：脂肪过高 + 乳化/均质不足。
*   **解法**：加强均质；控制总脂肪；避免“粉+酱+块”三叠。

## 附录：研发记录模板 (T9)

每次测试务必记录，以便追溯：
1.  **配方版本**：[口味]-[路线]-[日期]-[版本]-[变更点]
2.  **工艺批次**：过筛Y/N、做浆Y/N、均质方式、老化时间、上机温度。
3.  **机器参数**：出杯温度、Overrun、连续出杯漂移。
4.  **感官测试 (1-5分)**：细腻度、香气、甜腻、挺立(3min)、融化(10min)、残留。
