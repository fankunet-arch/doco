# 00_总索引与交付清单（整理版 v1.7）

## 0. 本项目一句话目标
做一套“类 llaollao 的门店模型”，产品走 **Soft-serve 软冰**，主攻 **抹茶 / 巧克力** 两大体系，并能稳定承载 toppings，实现：

- **细腻**：无冰渣/无颗粒
- **挺立**：堆得高、融化慢、少出水
- **风味清晰**：抹茶香/可可香穿透，不苦不腻

> 以上三点 = 全项目“总成功标准”。后续所有配方/设备/工艺/运营都为它服务。

## 2. 学习路线（两条线并行：M 系列 / T 系列）

### 2.1 M 系列（门店完整课程：从认知→产品→设备→门店→运营→扩张）
- M0｜入门地图：冰淇淋行业到底分几类（gelato / ice cream / sorbet / granita / frozen yogurt / soft serve）
- M1｜产品基础：口感、结构、配方逻辑
- M2｜原料体系：从“买什么”到“怎么控成本”
- M3｜口味设计：简单但可复购
- M4｜设备课：买的不是机器，是稳定出品能力
- M5｜工艺与SOP：把手艺变标准
- M6｜门店模型：坪效、动线、翻台（概览）
- M7｜菜单与定价：利润率只是表象，关键是结构
- M8｜运营与营销：目标是稳定复购
- M9｜西班牙合规与卫生：不踩雷（概览）
- M10｜供应链与生产方式：自制 vs 半成品 vs 全成品
- M11｜财务模型：稳不稳、回本快不快
- M12｜复制与扩张：从一家到连锁要补的课
- **M13｜门店模型：坪效、动线、翻台（小店高吞吐｜深挖）** —— 对应文件 11
- **M14｜稳定复购运营 v1（不爆红也能稳｜深挖）** —— 对应文件 11
- **M15｜西班牙合规与卫生底座（软冰店高频雷区｜深挖）** —— 对应文件 11


### 2.2 T 系列（纯技术训练：每节课都能产出“可用结论”）
- T1：配方结构设计（搭骨架）
- T2：糖体系与冻结点（不更甜也能更软更细）
- T3：稳定剂/乳化剂体系（融化慢、挺立、不出水）
- T4：抹茶专项（分散、氧化、苦涩与香气）
- T5：巧克力专项（黏度、油感、塌陷与堵机）
- T6：机器参数与口感映射（温度/Overrun/负荷稳定性）
- T7：完整故障排查树（先查什么先改哪个旋钮）
- T8：设备选型深挖（按产品目标做选型决策表）
- T9：研发记录模板（版本/工艺/参数/评分/结论）
- T10：基础白奶软冰公式 v1（从零自制料液）

## 4. 作业索引（本文件只放 M0 作业；其他作业跟在各章节文件末尾）

### 作业 A｜选 2 家同类门店做快速观察（不需要进店坐）
记录下面 10 个数据（手机备忘录即可）：
1) 城市/商圈位置  
2) 门头风格（年轻/家庭/游客）  
3) 客单价区间（最便宜/最常见/最高）  
4) 杯型/克重逻辑（按尺寸？按重量？）  
5) topping 分类（水果/脆料/酱/谷物/冷冻料）  
6) 点单流程（先点杯型？先选 topping？）  
7) 高峰排队时：每单出杯大概多少秒  
8) topping 台是不是自选（自选损耗/卫生风险更高）  
9) 外卖做不做（看平台/店内提示）  
10) 你判断它“复购点”是什么（轻负担/活动/口味/社交）
