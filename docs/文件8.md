# T10｜基础白奶软冰公式 v1（从零自制料液）
## A）轻盈版（更像 llaollao 那种“清爽顺滑”，更适合西班牙大众）
**目标口感**：不腻、入口干净、回味奶香，适合做抹茶/巧克力基底
**配方（1 kg 料液）**
* 全脂牛奶（Asturiana entera）：**691 g**
* 奶油 35%（Nata para montar）：**129 g**
* 白砂糖：**100 g**
* 液态葡萄糖（Glucose syrup）：**60 g**
* Dextrosa（葡萄糖粉）：**20 g**
* （可选）稳定剂：**2 g（0.2%）** ← 没有也能先跑通，但稳定性会差一点
> 这套甜度不会齁甜，且更抗冰晶，适合你说的“中偏甜”。

## B）浓厚版（更“冰淇淋感”，更高级，但更容易腻）
**配方（1 kg 料液）**
* 全脂牛奶：**640 g**
* 奶油 35%：**200 g**
* 白砂糖：**95 g**
* 液态葡萄糖：**50 g**
* Dextrosa：**15 g**
* （可选）稳定剂：**2 g（0.2%）**

# 工艺流程（你先按这个标准动作做，成功率最高）
## 1）溶解/水化（建议做，不要偷懒）
1. 把 **牛奶 + 奶油** 倒锅里，加热到 **45–50°C**
2. 把 **白砂糖 + dextrosa** 先干混，再倒入搅拌溶解
3. 加入 **液态葡萄糖** 搅匀
4. 若有稳定剂：稳定剂先和少量糖干混再撒入，继续搅拌
5. 加热到 **65–70°C**，保持 **3–5 分钟**（不需要煮沸）
> 目的：彻底溶糖、让体系更细腻、稳定剂（如果有）能正确水化。
## 2）快速降温 + 熟成
* 迅速降到 **≤4°C**
* 冷藏 **4–12 小时**（强烈建议做，口感会“明显更软更顺”）
## 3）上机（软冰机）
* 料液温度：**2–5°C** 上机最稳
* 第一轮出品：先丢掉前面少量（机器状态未稳）
* 观察：是否“站得住”、是否“发虚”、是否“口感发涩/水”

# 你现在缺的关键点（我直接说结论）
* **没有专业稳定剂也能做出好吃的**，但：
  * 更容易化得快
  * 更容易“吃到后半段变粗/结冰晶”
* 如果你愿意专业化：后面我会让你加一个**最简单的软冰稳定剂方案**（0.2–0.35%区间），你就能从“能吃”变成“高级”。
---
明白：你要 **偏浓厚、绵密、零冰渣**，但“奶味不能压住巧克力/抹茶香”。这其实是一个“**浓厚靠结构，不靠堆奶油**”的配方方向——把**乳固体 + 糖体系 + 稳定/乳化 + 工艺**做到位，脂肪反而别太高。

# 1）你要的“浓厚但不盖香”基底（v1）
> 目标：口感厚、细、回口干净；奶香是底，不抢巧克力/抹茶
## 配方（1 kg 料液）
* 全脂牛奶：**610 g**
* 35%淡奶油（nata para montar）：**230 g**
* 白砂糖：**85 g**
* 液态葡萄糖（glucose syrup）：**55 g**
* Dextrosa：**18 g**
* 稳定剂（如果你用 Sosa “Pro espuma frío” 这种）：**2.5 g（0.25%）**
**为什么这套更符合你要的：**
* 脂肪大概落在 **8–9%**（厚但不“奶油糊嘴”，不会压香）
* 用“葡萄糖浆 + dextrosa”把**冰点压下去** → 口感更软、更不结渣
* 0.25% 稳定剂让**水分被抓住** → 细腻、抗融、抗冰晶
---
# 2）关键工艺（决定你有没有“冰渣”）
你想要“零颗粒”，工艺比配方还重要：
1. **加热到 82–85°C**（别只到70）
   * 目的：让蛋白更“撑结构”，口感明显更绵密
2. **立刻降温到 ≤4°C**
3. **冷藏熟成 8–12 小时**（至少 6 小时）
   * 这是“细腻感”的核心步骤
4. 上机前料液温度 **2–5°C**
5. **第一轮出品丢掉一点**（机器未稳时最容易粗、最容易有渣）
---
# 3）“不盖住巧克力/抹茶香”的三条硬规则
1. **别用带强风味的乳品做基底**
* 暂时别用：dulce de leche、香草糖、crème fraîche（会抢风味/改变酸感）
2. **脂肪别再往上堆**
* 你想更厚，优先加“结构”而不是加奶油
* 奶油太高会把抹茶香“糊住”，巧克力也会变“奶巧味”
3. **巧克力/抹茶要靠“香味载体”而不是靠糖压**
* 抹茶：用量够（一般 10–18g/kg 才有穿透力）
* 巧克力：可可粉负责“香气冲击”，巧克力豆负责“圆润和厚度”
---
# 4）下一步我建议你怎么试（最快出结果）
你直接按上面基底做 **两锅 1kg**：
* **锅A：只做白奶基底**（测试“绵密/零渣”是否达标）
* **锅B：白奶基底 + 可可粉 50g/kg**（先用可可粉看“香气穿透力”）
如果 **锅A** 都还有冰渣：不是巧克力问题，是 **糖体系/稳定剂/熟成/上机温度**问题，我会按现象给你“加哪一个、加多少”的修正。

# 1）你今天拍的这些产品：哪些值得买（按用途分）
## A. 做“软冰基底/结构”必备（建议直接选定一套长期用）
* **全脂UHT牛奶**：Central Lechera Asturiana / Président（两者都能用，但**最好统一一个品牌**做长期一致性）
* **淡奶油 35%（Nata para montar）**：
  * **Aro 35%**（性价比高、风味干净）✅
  * Puleva 35% ✅
  * Asturiana 38%（更厚但更容易“糊住香气”，除非你后面把用量降下来）⚠️
* **白砂糖 5kg（Aro）** ✅（做稳定的甜度与结构）
* **Glucose syrup（葡萄糖浆）** ✅（核心：压冰点、抗冰渣、提升绵密）
* **Dextrose（葡萄糖粉）** ✅（核心：进一步压冰点、让口感更“软”）
* **稳定剂**：你拍的 **Sosa Pro espuma frío** 可以当“临时方案”先试，但我更建议后续换成**专用 ice cream/soft-serve stabilizer（含E410/E412/E466 + 乳化剂E471）**，一致性会更稳。
## B. 巧克力体系（你要“顺滑偏甜不苦”）
* **优先买：Metro Chef 50% chips** ✅（甜感更友好、适合你说的方向）
* **可选加深层：Metro Chef 64% buttons（少量）** ✅（用一点点拉深度，不会苦）
* **不建议当主力：Cacao Barry Extra Brute 可可粉** ❌（太冲、太苦、容易盖抹茶）
* **可可粉（Valor 脱脂粉 16%）**：做“巧克力香气穿透”很好，但要控制用量 ✅
## C. 只适合做 topping / 辅料（不要拿来做基底主结构）
* **Dulce de leche（罐头/挤压瓶）**：做浇酱/旋纹✅（别进主料液，不然奶香+焦糖会抢抹茶/巧克力）
* **Oreo crumbs / Lotus crumbs**：做脆料✅
* **Isomalt / Glace sugar / 香草糖 / 液体代糖**：对你这个“软冰基底”价值不大，先别买或别作为主路径⚠️

# 2）总部“半成品”方案（最适合你：粉体预拌，门店只加牛奶/淡奶油）
你要跨门店一致、操作最少、物流最省：**做成“预拌粉（Premix Powder）”** 是最稳的。
## 总部做 2 个SKU（粉体）
* **SKU-BASE：基底结构粉（白底）**
  含：糖 + dextrose + 稳定剂/乳化剂 + 脱脂奶粉（提升乳固体/绵密）
* **SKU-CHOCO：巧克力风味粉**（做你要的“顺滑偏甜”）
  含：可可粉（用温和的）+ 少量香草/盐（可选）
  *如果你追求更“高级的巧克力圆润感”，再额外配一个“50%巧克力豆小包”（门店做1个小动作融化即可，见下面“升级版”）*
---
# 3）门店标准化配比（兼容你还没定机型）
先给你一个“**通用浓厚但不盖香**”的出厂口感基准。门店只称重、搅拌、冷藏熟成即可。
## 3.1 白底（1 kg 料液）
* UHT全脂牛奶：**680 g**
* 35%淡奶油：**180 g**
* **SKU-BASE 基底结构粉：140 g**
> 这套脂肪大概在 **8.5–9%**：够厚、够绵密，但不会奶油糊嘴、不会压抹茶/巧克力香。
## 3.2 巧克力（1 kg 料液，顺滑偏甜路线）
* 按上面白底 1 kg
* 另外加入：**SKU-CHOCO 巧克力风味粉 25–35 g / kg**
> 先从 **30g/kg** 起步。你要“不苦”，就别一上来可可粉拉太高。
### 升级版（更高级、更圆润，但门店多一步）
* 仍按白底 1 kg
* 门店取其中 **80g 牛奶加热到 60–65°C**（不用煮）
* 融化 **50%巧克力豆 40–60 g**，倒回搅匀
* 同时可可粉量降到 **15–20 g/kg**（香气穿透 + 圆润感更像“真巧克力”）
---
# 4）为“零冰渣”必须统一的SOP（这决定成败）
门店只要照做，你的“绵密无颗粒”成功率会非常高：
1. **搅拌分散**：先用少量牛奶把粉打成浆，再加剩余液体
2. **冷藏熟成 8–12小时（至少6小时）**
3. 上机温度 **2–5°C**
4. 稳定剂用量先按 **0.25%（2.5g/kg 成品料液）** 的级别去做（如果你是SKU里已含，就按SKU剂量）
---
# 5）机型没定怎么办（重力 vs 泵机的“旋钮”）
你先按上面基准跑。等你机型定了，只改一个参数就能贴合：
* **如果是重力机（更怕太稠出不来）**：
  SKU-BASE 从 140g/kg → **130g/kg**
* **如果是打气泵机（更能吃稠，口感更“奶昔感”）**：
  SKU-BASE 从 140g/kg → **150g/kg**
（别动奶油比例太多；你要“香气不被盖”，脂肪别猛加。）
